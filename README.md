The **kmp-finance** project is a multiplatform project developed using the Kotlin programming language and the KMP (Kotlin Multiplatform) framework.

##### Available Platforms:

* Server (Beta)
* Android
* IOS
* Desktop
* Web

You can see OpenAPI documentation: [http://127.0.0.1:8080/openapi](http://127.0.0.1:8080/openapi)

You can see Swagger documentation: [http://127.0.0.1:8080/swagger](http://127.0.0.1:8080/swagger)
